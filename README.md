# AVLVisualizer
AVL Tree Visualizer using HTML, CSS, and JS



Features

Insert nodes

Self-balancing AVL logic

Dynamic tree drawing

Dark/Light theme toggle

Tech Stack

HTML – Web page structure (input, buttons, canvas)

CSS – Styling, layout, dark/light mode

JavaScript – AVL tree logic, insertions, rotations, and dynamic visualization

Canvas API – Drawing nodes, lines, and interactive tree

GitHub / GitHub Pages – Version control and hosting
